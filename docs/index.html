왜 소희인가

사장님은 본업에 집중하세요

소희는 한국 여성 이름의 AI 직원 페르소나입니다. 사장님이 직접 소희의 성격·말투·어휘를 설정하고, 소희가 사업장을 운영합니다.

😩

이런 적 있으시죠?

  • DM·예약·전화가 오면 본업이 자꾸 끊김
  • 콘텐츠 쓸 시간이 없어 SNS가 잠잠
  • 댓글 하나하나 답하기 벅참
  • 월말에 매출·성과 정리하느라 야근
🌸

소희가 대신합니다

사장님이 잠든 새벽에도, 영업 중인 낮에도, 쉬고 싶은 주말에도 — 소희는 쉬지 않고 응대하고, 발행하고, 기록합니다. 사장님이 설정한 페르소나로 일관되게.

"혼자 사업하면서 SNS 응대까지 하려니 지쳤는데,
소희한테 페르소나를 설정해두고 맡기니까, 본업에 집중할 수 있게 됐어요."
— 서울 마포구 네일숍 사장님
어떻게 작동하나

5분이면 소희 셋업 끝

복잡한 설정 없이, 사업장 자료만 올리면 소희가 나머지를 자동으로 배웁니다.

① 사업장 자료 업로드 → ② 소희 페르소나 설정 → ③ 채널 연결 → ④ 자동 응대 시작
📂

1. 사업장 자료 업로드

메뉴판, FAQ, 가격표, 서비스 안내 PDF를 그대로 올리세요. 소희가 자동으로 학습합니다.

🎭

2. 소희 페르소나 설정

이름(소희 → 자유 변경), 톤, 어투, 금지어, 인계 규칙을 사장님이 직접 정의합니다.

🔌

3. 채널 연결

Instagram, Threads, Mastodon, 카카오 채널, 네이버 플레이스 — 클릭 한 번으로 연결.

💬

4. 자동 응대

DM·댓글·예약 문의를 소희가 1차 응대. 위험하거나 중요한 건 사장님께 인계.

📊

5. 주간 리포트

매주 월요일, 지난주 응대·발행·매출·방문자 통계를 요약해 사장님께 보고.

📝

6. 콘텐츠 자동 생성

주간 자동 일정에 따라 초안을 작성하고 검수 채널에 올립니다. 사장님은 검수만 하세요.

소희의 페르소나

사장님이 직접 만듭니다

소희는 처음에 표준 한국 여성 페르소나로 시작하지만, 사장님이 자신의 사업에 맞게 자유롭게 변경할 수 있습니다.

🎀

이름

기본 "소희"에서 "지원이", "민호", "도우미" 등 자유롭게 변경 가능. 브랜드와 어울리는 이름을 지어주세요.

🗣️

말투와 존칭

"습니다 체"부터 "요 체", "반말"까지. 카페는 캐주얼하게, 컨설팅은 격식 있게 — 사장님이 정합니다.

📚

추천 어휘

"오늘의 추천", "시즌 한정", "사장님 직접 로스팅" 등 사장님만의 어휘를 등록하면 소희가 일관되게 사용합니다.

🚫

금지어

"싸게", "저렴한" 같은 단어를 등록하면 소희는 절대 사용하지 않습니다. 브랜드 이미지 보호.

🤝

사람 인계 규칙

"환불", "민원", "가격협상" 같은 주제에는 소희가 자동으로 사장님께 인계합니다.

💬

메모리

"단골 김선영 씨는 아메리카노 2샷으로", "이번 주말에 휴업" — 사장님이 알려주면 소희가 기억합니다.

가격

가격 정책은 준비 중

지금 가입하시면 정식 가격 출시 전 무료 베타 기간 동안 모든 기능을 먼저 사용해보실 수 있습니다.

🚧

가격 정책 준비 중

정식 가격은 베타 사용자 피드백을 반영하여 결정됩니다.
사장님께 합리적인 요금으로 제공될 예정입니다.

FAQ

자주 묻는 질문

소희는 누구인가요?

소희는 사장님이 사용하는 AI 직원의 기본 페르소나 이름입니다. 한국 여성 이름으로, 사장님이 직접 다른 이름으로 변경할 수 있습니다. (예: 지원이, 도우미, 반장 등)

설치가 복잡한가요?

아니요. 사업장 자료 업로드 → 페르소나 설정 → 채널 연결까지 평균 5분이면 충분합니다. 전 과정에서 가이드가 안내합니다.

소희가 잘못된 답변을 하면 어떡해요?

모든 자동 응대는 사장님이 미리 정의한 어휘·금지어·톤 가이드 안에서만 동작합니다. 가격이 궁금하다거나 민감한 문의는 자동으로 사장님께 인계되어 직접 응대할 수 있습니다.

내 사업장 데이터는 안전한가요?

네. 모든 데이터는 사업자별로 완전 분리되어 저장됩니다. Active Record Encryption으로 민감 정보는 자동 암호화됩니다.

어떤 업종에 잘 맞나요?

미용실, 네일숍, 피부관리실, 카페, 음식점, 소매점, 학원, 클리닉 등 — FAQ와 응대 패턴이 정형화된 소상공인 사업장에 특히 잘 맞습니다.

언제든 해지할 수 있나요?

네, 해지 수수료 없습니다. 설정한 자동화도 즉시 중단되며, 데이터는 해지 후 30일간 백업 가능합니다.
